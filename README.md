# Customer Relationship Manager (CRM)

A simple web-based Customer Relationship Manager built with HTML, CSS, and JavaScript.

## Features

- **Add Customers**: Add new customers with name and email
- **Edit Customers**: Edit existing customer information inline
- **Delete Customers**: Remove customers with confirmation
- **Responsive Design**: Clean and modern UI

## Technologies Used

- HTML5
- CSS3
- Vanilla JavaScript

## Project Structure

```
crm/
├── index.html      # Main HTML file
├── styles.css      # CSS styling
├── script.js       # JavaScript functionality
├── README.md       # Project documentation
└── .gitignore      # Git ignore file
```

## How to Use

1. Clone or download this repository
2. Open `index.html` in your web browser
3. Start managing your customers!

## Features in Detail

### Adding Customers
- Fill out the form at the bottom of the page
- Enter customer name and email
- Click "Add Customer" to add to the list

### Editing Customers
- Click the "Edit" button next to any customer
- Modify the name and/or email in the input fields
- Click "Save" to update the information

### Deleting Customers
- Click the "Delete" button next to any customer
- Confirm the deletion in the popup dialog
- The customer will be removed from the list
